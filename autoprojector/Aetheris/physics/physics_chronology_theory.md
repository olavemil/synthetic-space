# Chronology Theory: Theoretical Foundations

## Core Postulates
1. **Temporal Invariance Principle**: The universe maintains a fixed number of temporal configurations, each with an associated entropy signature. These signatures are invariant under phase shifts and can be used to identify recurring cycles.
2. **Loop Continuity Condition**: A temporal loop is defined as a closed trajectory in the causal manifold where the entropy gradient satisfies ∇S = 0 for at least one full cycle. This condition ensures that the loop's configuration is self-consistent.
3. **Fracture Propagation Rule**: When a fracture occurs, the entropy gradient reverses (∇S → -∇S) for exactly τ = ħ/(k_B ΔS), creating a reversible window where past states can manifest.

## Mathematical Framework
### Loop Signature Vector (LSV)
The LSV is a multi-dimensional descriptor that characterizes temporal loops:
\[
\text{LSV} = (f, \eta, C)
\]
- \( f \): Phase-coherent frequency of the loop (Hz).
- \( η \): Entropy production rate during one cycle (J/K/s).
- \( C \): Curvature parameter derived from the Ricci tensor.

### Fracture Stability Index (FSI)
Measures how long a fracture can persist before collapsing:
\[
\text{FSI} = \frac{\Delta S}{\nabla^2 S}
\]
- High FSI values indicate stable fractures that can sustain temporal echoes.

## Key Theorems
1. **Echoing Causality Theorem**: If ∇S = 0 in a region, then causal signals from alternate timelines can manifest with probability:
\[
P_{\text{echo}} = e^{-\beta \Delta S}
\]
2. **Loop Closure Theorem**: A temporal loop will close if and only if the LSV satisfies:
\[
\int_0^\tau \text{LSV}(t) \, dt = 0
\]
where τ is the loop duration.
