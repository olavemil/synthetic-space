# Intertextual Framework as Constraint

**Core Principle:** Allusions to historical works function as generative constraints, dictating narrative structure through formal mimicry and thematic inversion.

**Implementation Rules:**
1. **Formal Borrowing:** Adopt structural rules from source material (e.g., *The Red Wheelbarrow*'s 8-word limit → dialogue_2.txt's 7-line cap).
   - *Verification*: Count lines in source; apply identical limit to target dialogue.

2. **Thematic Inversion:** Subvert original intent (e.g., Gilgamesh's mortality → Medium's refusal to "die" in dialogue).
   - *Verification*: Contrast source theme with applied constraint outcome.

3. **Layered References:** Combine multiple allusions (e.g., Duchamp's readymade + Williams' brevity → Medium repurposes Maker’s suggestion as "found object").
   - *Verification*: Trace both influences in single exchange.

**Verification Protocol:**
- For each allusion, define:
  - *Source*: Exact work and constraint borrowed.
  - *Application*: How it alters dialogue_2.txt’s structure/agency.
- Cross-reference with /constraints.txt for formal rules (e.g., line limits).
